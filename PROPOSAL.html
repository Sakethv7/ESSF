Ek Soach Saathiya Foundation — Website Project

Work Summary & Proposal

📅 May 2026 🌐 eksoach.in 👁 Preview: sakethv7.github.io/ESSF

1 — What Has Been Done ✓ COMPLETE

🖥️
5-Page Custom Website

Home, Executives, Past Events, Social Activity, Contact — built from scratch with no paid templates or frameworks. Designed to load fast even on slow mobile connections.

📱
Fully Responsive Design

Tested and working on iPhone, Android, iPad, MacBook, and desktop. The navigation collapses into a hamburger menu on phones and tablets with a properly sized tap target for iOS.

🖼️
221 Photos Compressed & Organised

All photos were compressed from several GBs down to ~6MB total without visible quality loss. Organised into dated event folders. Past Events shows 8 photos at a time with a "Show all" button so the page doesn't require endless scrolling.

🎠
Hero Slider with Full Swipe Support

4-slide banner that auto-advances every 5 seconds. Works with: tap on arrows, dot buttons, finger swipe on mobile, two-finger trackpad swipe on MacBook, and click-and-drag on desktop.

⚙️
Automated Deployment Pipeline

Every code push automatically updates the site so visitors always see the latest version — no manual steps needed. A one-command deploy script packages everything for upload to the production server.

🔒
Security & Performance

Full security review completed. Production server configured with: automatic HTTPS redirect, compressed file delivery (gzip), long-term browser caching for fast repeat visits, and standard security response headers.

🐛
Testing, Bug Fixes & Accessibility

iOS hamburger menu fixed, caching issues resolved, slider overflow corrected. All interactive elements labelled for screen readers. Tested across iPhone Safari, Chrome Android, MacBook, and desktop.

✅ Website is live at sakethv7.github.io/ESSF — fully functional, tested, and ready for client review before going live on eksoach.in.

4 — Future Enhancements ON REQUEST

These can be added any time after the site goes live, one at a time.

🎖️ Impact / Numbers Section

A strip on the home page showing key achievements — total students supported, amount donated, events held, years active. Numbers provided by Monty. Very effective for first-time visitors and potential donors.

Quick to build
💬 Testimonials Section

Quotes from scholarship recipients, students, or parents about the impact ESSF has had. Displayed as cards with a photo and a short quote. One of the most powerful trust-builders on any NGO website.

Quick to buildHigh impact
🏦 Donation / Bank Details Page

A dedicated page showing UPI ID, bank account number, IFSC code, and optionally a QR code — for anyone who wants to donate to the foundation. Currently there is no donation information on the site.

Quick to build
🐦 X (Twitter) Feed

Once the ESSF X account is created, a live feed of recent posts can be embedded on the home page or Social Activity page. Visitors see the latest updates without leaving the website. Requires an active X account first.

Quick to buildGood for engagement
📩 Contact Form

Currently the Contact page shows the ESSF email address. A contact form means visitors can type their name, email, and message directly on the website and it gets delivered to the ESSF inbox — without needing to open their own email app. Works well for parents, donors, or media enquiries.

Quick to build
🗺️ Interactive Map on Contact Page

Currently the Contact page has a "Open in Google Maps" link. An embedded map would show the location visually right on the page without needing to click away. The current link is functional — this is a cosmetic upgrade.

Quick to build
🌐 Hindi Language Support

Adds a language toggle so visitors can switch the website between English and Hindi. Hindi translations of all content would need to be provided by Monty. Recommended once the English version is stable and approved.

Medium effort
📅 New Event Pages

Each year when a new event is held, photos can be compressed and added to the Past Events gallery. This is an ongoing content update, not a feature build — covered under the maintenance plans above.

Part of maintenance
🗂️ Self-Edit Panel (CMS)

A "Content Management System" gives Monty an admin login where he can update text, add photos, and change content himself — without needing a developer. This requires rebuilding parts of the site and is a bigger investment. Not needed now, but a good long-term option if the foundation grows.
