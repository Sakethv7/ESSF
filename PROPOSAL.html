1 Work Delivered — Initial Build

🖥️
5-Page Custom Website

Built from scratch — Home, Executives, Past Events, Social Activity, Contact. Pure HTML/CSS/JS, no frameworks. Designed to load fast even on slow mobile connections. Includes hero slider, activity cards, photo gallery, and footer.

₹15,000–20,000core build
📱
Fully Responsive Design

Works on all screen sizes — iPhone, Android, iPad, laptop, and desktop. Separate layouts for mobile (≤480px), tablet (≤900px), and desktop. Navigation becomes a hamburger menu on smaller screens with a fixed iOS-compatible touch target.

Includedin core build
🖼️
221 Photos — Compressed & Organised

All photos received were compressed to optimised thumbnails (~25KB each) using a Python script, reducing total size from several GBs to ~6MB. Organised into dated event folders. Past Events page shows first 8 photos per event with a "Show all" button to avoid endless scrolling.

₹3,000–5,000photo management
🎠
Hero Slider — Touch, Swipe & Trackpad

4-slide hero banner with auto-advance every 5 seconds. Supports: tap on arrow buttons, dot navigation, finger swipe on mobile, two-finger trackpad swipe on MacBook, and click-and-drag on desktop mouse.

Includedin core build
⚙️
CI/CD Pipeline & Deploy Tooling

GitHub Actions workflow that automatically updates cache-busting version strings on every code push, so visitors always get the latest CSS and JS. One-command deploy script (deploy.sh) that packages the site for FTP upload to the production server — strips dev files, injects version strings, outputs a clean folder ready to upload.

₹3,000–5,000devops tooling
🔒
Security Audit & Hardening

Full security review — checked all external links for proper rel="noopener noreferrer", verified no XSS risks, no inline event handlers, no exposed credentials. Added Apache .htaccess with HTTPS redirect, gzip compression, browser cache headers (1 year for assets), and security response headers (X-Frame-Options, X-Content-Type-Options, Referrer-Policy).

₹3,000–5,000security + devops
♿
Accessibility Fixes

ARIA labels on all interactive elements (slider dots, show-more button, hamburger menu). Keyboard-navigable nav. Proper alt text on all images. Semantic HTML structure throughout. Screen reader compatible.

Includedin iterations
🐛
Iterations, Debugging & Testing

iOS hamburger menu fix (tap target size, z-index, position:fixed dropdown), browser cache-busting across 3 rounds of deployment, slider overflow fix, nav logo responsive sizing across 3 breakpoints, tagline strip styling consistency, gallery collapse feature, trackpad swipe detection.

₹3,000–5,000QA & fixes
